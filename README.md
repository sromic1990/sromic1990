## Building

[**Dark Chocolate**](https://github.com/sromic1990/Dark-Chocolate) — A multi-backend hardware abstraction layer in C++, targeting Vulkan, DX12, and Metal behind a unified interface. The goal is a clean, minimal abstraction that maps closely to how modern GPUs actually work — no hidden magic, no legacy baggage.
